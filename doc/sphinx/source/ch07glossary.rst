********
Glossary
********

.. glossary::

   AVX
      Advanced Vector Extensions

   binary64
      IEEE 754-2008 double precision

   C11
      C programming language standard ISO/IEC 9899:2011

   C++11
      C++ programming language standard ISO/IEC 14882:2011

   CPU
      Central Processing Unit

   FMA
      Fused Multiply-Add

   FLOP
      Floating-point Operation

   FPU
      Floating-Point Unit

   GCC
      GNU Compiler Collection http://gcc.gnu.org

   GLIBC
      GNU C Library https://www.gnu.org/software/libc

   LIBSTDC++
      GNU Standard C++ Library http://gcc.gnu.org/libstdc++

   MXCSR
      Media eXtension Control and Status Register

   IEEE 754-1985
      IEEE Standard for Binary Floating-Point Arithmetic (also known as IEC
      60559:1989)

   IEEE 754-2008
      IEEE Standard for Floating-Point Arithmetic

   MMX
      Floating-Point instruction set

   SSE
      Streaming SIMD Extensions

   SIMD
      Single Instruction Multiple Data
